# Fluent UI React v9 Sandbox

A sandbox web app demonstrating Fluent UI React v9 components.

## Features

- **Demo Page 1**: All Fluent UI components displayed in a centered table
- **Demo Page 2**: Microsoft Account page layout recreation using Fluent UI components

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown in the terminal (typically http://localhost:5173)

## Technologies

- React 18
- Fluent UI React v9
- TypeScript
- Vite
- React Router
